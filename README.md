# A TMDB App

Made with Flutter

## API Used

| Name   | Link                              |
|:-------|:----------------------------------|
| `TMDB` | https://developer.themoviedb.org/ |

## Tech Stack

- Clean Architecture
    - Presentation
    - Domain -> Use-Case
    - Data -> Local DB / API (Dio)
- Go Router
- Dependency Injection
- Intl (l10n)
- Bloc for state management
- Hive/Secure Storage
- Mockito

## Running the app

Flutter run --dart-define=api_key=`<Your-tmdb-api-key>`
